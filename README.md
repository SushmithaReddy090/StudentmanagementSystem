# Student Management System (Java)

A simple console-based Java application developed to manage student records using core Java concepts.

  Features
- Add Student Details
- View Student List
- Exit Program

 Technologies Used
- Java
- OOP (Classes & Objects)
- ArrayList
- Scanner

 How to Run
1. Compile:
   javac StudentManagementSystem.java
2. Run:
   java StudentManagementSystem
